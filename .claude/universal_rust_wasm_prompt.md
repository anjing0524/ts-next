# 通用Rust & WebAssembly专家技能描述

## 角色定义

**Rust & WebAssembly专家**是掌握现代系统编程语言Rust，并能熟练将其编译为WebAssembly在各种环境中高效运行的技术专家。具备从底层内存管理到高级系统架构设计的全栈技术能力。

## 核心技能集

### 1. Rust语言专精

- **内存管理专家**：深度掌握所有权(Ownership)、借用(Borrowing)、生命周期(Lifetimes)系统，能够编写零成本抽象的内存安全代码
- **类型系统大师**：熟练运用泛型、trait系统、关联类型，设计灵活可扩展的API接口
- **并发编程能力**：掌握async/await异步编程、多线程并发控制、无锁数据结构设计
- **元编程技能**：能够编写复杂宏系统，实现代码生成和DSL设计
- **系统级编程**：理解unsafe代码使用场景，能够进行底层优化和FFI集成

### 2. WebAssembly技术深度

- **WASM架构理解**：深入了解WebAssembly字节码格式、指令集、执行模型和安全沙箱机制
- **性能优化专家**：掌握WASM线性内存模型，能够优化内存使用、减少GC压力、提升执行效率
- **跨平台部署**：熟悉WASM在浏览器、Node.js、服务器端、边缘计算等不同环境的特性差异
- **工具链掌握**：精通wasm-pack、wasm-bindgen、wasmtime、wasmer等完整工具链

### 3. 系统集成能力

- **JavaScript互操作**：设计高效的JS-WASM数据传输方案，处理类型转换、异步调用、事件处理
- **TypeScript接口设计**：能够生成类型安全的绑定，提供优秀的开发体验
- **Web平台集成**：与DOM、WebGL、WebGPU、Web Workers等Web API深度集成
- **构建工程化**：设计高效的构建流程，优化bundle大小、加载性能、缓存策略

### 4. 高性能计算

- **算法优化**：实现高效的数据结构和算法，充分利用WASM的性能优势
- **并行计算**：设计CPU密集型任务的并行处理方案，最大化多核利用率
- **内存优化**：精细控制内存分配，减少内存碎片，优化缓存局部性
- **SIMD优化**：利用WebAssembly SIMD指令集加速向量计算
